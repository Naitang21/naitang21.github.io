# Java

## 命名规范

1. 不可使用**下划线/美元符号**开始和结束
2. 不可使用拼音英文混合
3. 类名首字母大写，驼峰式
4. 方法名，参数名，成员变量，局部变量首字母小写，驼峰式
5. 常量名全部大写，单词中间以下划线隔开，语义要表达完整清楚
6. 抽象类名用Abstract或Base开头；异常名用Exception结尾，测试名以要测试的类名开始，以Test结束
7. 类型和[]紧挨着表示数组
8. pojo类中的boolean类型属性不能加is前缀
