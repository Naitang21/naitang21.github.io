如果记得用户名，可以进到进入自己的GitHub主页，进入某一个仓库，点进`commit`页面，此时的url形如：
```
https://github.com/用户名/项目名/commit/一个字符串
```
在这个url后面加上`.patch`，然后在`From:xxxxxx`就可以看到自己的邮箱了
