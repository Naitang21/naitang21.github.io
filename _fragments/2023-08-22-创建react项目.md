## 创建react项目

```
npx create-react-app projectName
```

## 启动react项目

```
npm start
```

## redux

### 添加redux依赖

```
npm install @reduxjs/toolkit
```

查看是否添加依赖成功：在package.json中查看是否存在依赖：

```
"react-redux": "^8.1.1",
"react-scripts": "5.0.1",
```

## router

参考资料：[React Router](https://reactrouter.com/en/main/start/tutorial)

### 添加依赖

```
npm install react-router-dom
```
