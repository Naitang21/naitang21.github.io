Java Clock是Java8新特性之一，Clock属于`java.time`包，提供了使用时区和最佳可用系统时钟访问当前的瞬间，日期和时间。
可以使用`Clock`代替`System.currentTimeMillis()`和`TimeZone.getDefault()`方法

**和Java 9的区别**：Java 8不带纳秒，Java 9带纳秒
